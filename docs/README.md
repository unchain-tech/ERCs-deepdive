# EIPs-Encyclopedia

## About

OpenZeppelin の主要規格について，日本語ベース・コードベースでまとめていこうという企画です.
また、ERC 規格毎によくある bug の対策サンプルなどもまとめています.

全ての Solidity 初学者・非修得者に愛をこめて.
